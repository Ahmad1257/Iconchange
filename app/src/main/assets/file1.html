Q#:1.1. Express the following quantities using prefixes?

(a) 5000g (b) 2000000W (c) 52 X 10-10kg (d) 225 X 10-8s

Ans. Solution:

(a) 5000g

= 5 X 103g                   (∴ 103g= 1kg)

= 5kg

(b) 2000000W

= 2 X 106W                 (∴ 106W= 1MW)

= 2MW

(c) 52 X 10-10kg

= 52 X 10-10 X 103g   (∴ 1kg =103g)

=52 X 10-7g

=5.2 X 101X 10-7g

= 5.2 X 10-6 g             (∴ 10-6 g = 1ùúág)

(d) 225 X 10-8 s

= 2.25 X 102 X10-8s

= 2.25 X 10-6 s          (∴ 10-6s = 1ùúás)

= 2.25ùúás

Q#:1.2. How do the prefixes micro, nano and pico relate to each other?

Ans. Solution:

As,

micro = 10-6, nano = 10-9, pico = 10-12

So,

1000 nano = 1000 X 10-9

= 103 X 10-9  =10-6

    1000 nano= 1 micro

Also,

1000 pico = 1000 X 10-12

= 103 X 10-12 = 10-9

    1000 pico = 1nano

Q#:1.3. Yoru hair grow at the rate of 1mm per day. Find their growth rate in nms-1?

Ans. Solution:

Length of hair = 1m = 1 X 10-3m

Time = 24 hr = 86400sec

To Find:

Growth rate per day = ?

Formula:

Growth rate per day = Length of hair/Time

Solution:

Putting the values, we have

Growth rate per day = 1X10−3m/ 86400sec

= 1.157 X 10-8 ms-1

= 11.57 X 10-9 ms-1

= 11.57 nms-1           (∴ 1n = 10-9)

Result:

The growth rate per day of human hair is 11.57 nms-1

Q#:1.4. Rewrite the following in standard form?

(a) 1168 X 10-27              (b) 32 X 105

(c) 725 X 10-5 kg             (d) 0.02 X 10-8

Ans. Solution:

(a) 1168 X 10-27

= 1.168 X 103 X 10-27

= 1.168 X 103-27

= 1.168 X 10-24

(b) 32 X 105

= 3.2 X 101 X 105

= 3.2 X 101+5

= 3.2 X 106

(c) 725 X 10-5 kg

= 7.25 X 102 X 10-5 kg

= 7.25 X 102-5 kg

= 7.25 X 10-3 kg                   (∴ kg = 103g)

= 7.25 X 10-3 X 103 g

= 7.25 X 10-3+3 g

= 7.25g

(d) 0.02 X 10-8

=2.0 X 10-2 X 10-8

=2.0 X 10-2-8

=2 X 10-10

Q#:1.6. One closing the jaws of Vernier caliper, zero the Vernier scale is on the right to its main scale such that 4th division of its Vernier scale coincides with one of the main scale division. Find its zero error and zero correction?

Ans. Solution:

Vernier division coinciding with main scale = 4th division

Least count = 0.01 cm

Vernier scale reading = 4 X 0.01

                               = 0.04cm

Since zero of Vernier scale is on the right side of zero of the main scale, so, instrument has measured more than actual reading. It is positive zero error and correction should be negative.

Zero error = +0.04cm

Zero correction = -0.04cm

Q#:1.10. A chocolate wrapper is 6.7cm long and 5.4cm wide. Calculate its area up to reasonable number of significant figures?

Ans. Solution:

Data:

Length of wrapper = 6.7cm

Width of wrapper = 5.4cm

Required:

Area of wrapper = ?

Solution:

Area = length X width

= 6.7cm X 5.4cm

= 36.18cm2

According to the principle of reasonable numbers of significant figures.

Area = 36cm2 (after rounding)
